<objective>
Bulk load all 403K JHSC rows into jhsc_tweets table using chunked inserts with tqdm progress. Script is idempotent. Observatory queries become possible.
</objective>

## Context

JHSC CSV files in `mizan/backend/data/jhsc/` (mounted as `/app/data/jhsc` in Docker).
- `annotated-hatetweets-4-classes_train.csv` — 302,766 rows
- `annotated-hatetweets-4-classes_test.csv` — 100,923 rows
- Columns: `tweet_id`, `new_tweet_content`, `Label`
- Labels (all lowercase): `negative`, `neutral`, `positive`, `very positive`
- NO date column — tweet_year/tweet_month will be null

Working directory: `/Users/ahmadalzaro/Desktop/AI Hatespeech Project/`

## Tasks

<task id="1" name="Create seed_jhsc.py">
Create `mizan/backend/scripts/seed_jhsc.py` that:

1. Checks if `jhsc_tweets` already has rows — exits early if so
2. Reads both train and test CSV files in chunks of 10,000 rows
3. For each chunk, builds list of dicts and bulk inserts using `pg_insert().on_conflict_do_nothing(index_elements=["id"])`
4. Shows tqdm progress bar
5. Prints final breakdown by label

Key: Use `from sqlalchemy.dialects.postgresql import insert as pg_insert` for ON CONFLICT DO NOTHING.
Valid labels set: `{"negative", "neutral", "positive", "very positive"}`
</task>

<task id="2" name="Run JHSC load">
```bash
docker compose -f mizan/docker-compose.yml exec backend python scripts/seed_jhsc.py
```
Expected: ~403K rows loaded in ~1-2 minutes.
</task>

<task id="3" name="Verify JHSC load">
```bash
docker compose -f mizan/docker-compose.yml exec db psql -U mizan -d mizan -c "SELECT COUNT(*) FROM jhsc_tweets;"
docker compose -f mizan/docker-compose.yml exec db psql -U mizan -d mizan -c "SELECT label, COUNT(*) FROM jhsc_tweets GROUP BY label ORDER BY label;"
```
Expected: ~403K total, 4 label values present.
</task>
