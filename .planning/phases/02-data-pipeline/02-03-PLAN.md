## Context

Phase 2 Plan 01 created the `content_examples` table with `ContentLabel` and `HateType` enums.
All dataset files are in `mizan/backend/data/` (mounted as `/app/data` in Docker).

**Label mappings:**
- JHSC: `negative` → hate (unknown), others → not_hate
- Let-Mi: `category != none` → hate (gender), `none` → not_hate
- MLMA: `hateful`/`abusive_hateful`/`offensive_hateful` → hate; `normal` → not_hate; target → hate_type
- AJ: `languagecomment == -2` (conf ≥ 0.75) → hate; `-1` → not_hate; drop `0`

**Existing models** (in `mizan/backend/app/models/content_example.py`):
- `ContentExample` — ORM model
- `ContentLabel` — enum: hate, offensive, not_hate, spam
- `HateType` — enum: race, religion, ideology, gender, disability, social_class, tribalism, refugee_related, political_affiliation, unknown

## Tasks

<task id="1" name="Create seed_content.py">
Create `mizan/backend/scripts/seed_content.py` that:

1. Checks if `content_examples` already has rows — exits early if so (idempotent)
2. Loads ~125 rows from JHSC:
   - Read `data/jhsc/annotated-hatetweets-4-classes_train.csv`
   - Columns: `tweet_id`, `new_tweet_content`, `Label`
   - Map `negative` → (hate, unknown); others → (not_hate, null)
   - Sample 125 rows with random_state=42
   - dialect="jordanian", source_dataset="jhsc"
3. Loads ~125 rows from Let-Mi:
   - Read `data/let-mi/let-mi_train_part.csv`
   - Columns: `text`, `category`, `misogyny`, `target`
   - Map `misogyny != "none"` → (hate, gender); `none` → (not_hate, null)
   - Sample 125 rows with random_state=42
   - dialect="levantine", source_dataset="let_mi"
4. Loads ~125 rows from MLMA:
   - Read `data/mlma/ar_dataset.csv`
   - Columns: `HITId`, `tweet`, `sentiment`, `directness`, `annotator_sentiment`, `target`, `group`
   - Map sentiment containing "hateful" → hate; "normal" → not_hate; "offensive" → offensive
   - Map target: "gender" → gender, "religion" → religion, "origin" → race, "disability" → disability, "other" → unknown
   - Sample 125 rows with random_state=42
   - dialect="mixed", source_dataset="mlma"
5. Loads AJ hate examples:
   - Read `data/aj-comments/AJCommentsClassification-CF.xlsx` using openpyxl
   - Filter: `languagecomment == -2` AND `languagecomment:confidence >= 0.75` → hate
   - Also sample some `languagecomment == -1` rows for not_hate balance
   - Target: ~125 total (as many hate as available, fill rest with not_hate)
   - dialect="mixed", source_dataset="aj_comments"
6. Bulk inserts all rows using `sqlalchemy.insert` with `execution_options(render_nulls=True)`
7. Prints breakdown by source and label
