<task id="2" name="Create data README">
Create `mizan/backend/data/README.md` documenting all dataset sources, licenses, column formats, and label mappings to the Mizan schema.

Include sections for:

**JHSC** — Jordanian Hate Speech Corpus
- Files: `annotated-hatetweets-4-classes_train.csv`, `annotated-hatetweets-4-classes_test.csv`
- Columns: `tweet_id`, `new_tweet_content`, `Label`
- Labels: `negative` (hate), `neutral`/`positive`/`very positive` (not_hate)
- License: CC BY 4.0
- Citation: Ahmad et al. 2024, Frontiers in AI

**Let-Mi** — Levantine Misogyny Dataset
- File: `let-mi_train_part.csv`
- Columns: `text`, `category`, `misogyny`, `target`
- Labels: `misogyny` != `none` → hate (gender), `none` → not_hate
- Citation: Mulki & Ghanem, WANLP 2021

**MLMA Arabic** — Multilingual Multi-Aspect Hate Speech
- File: `ar_dataset.csv`
- Columns: `HITId`, `tweet`, `sentiment`, `directness`, `annotator_sentiment`, `target`, `group`
- Labels: `hateful`/`abusive_hateful` → hate; `normal` → not_hate; target maps to hate_type
- Citation: Ousidhoum et al. 2019

**AJ Comments** — Al Jazeera Arabic Comments (CrowdFlower)
- File: `AJCommentsClassification-CF.xlsx`
- Columns: `body`, `languagecomment`, `languagecomment:confidence`
- Labels: `-2` (conf ≥ 0.75) → hate; `-1` → not_hate; `0` → drop (off-topic)
- 31,692 rows total; ~400 usable hate examples

**Arabic Religious** — Albadi 2018 (reserved)
- Tweet IDs only — AraHate lexicons for Phase 4
</task>
