# Plan 3.1 — Tailwind CSS Installation & RTL Layout Shell

## Goal
Install Tailwind CSS in the Vite frontend, configure it for Arabic RTL, and create a shared layout component that replaces the inline-styled header/nav pattern.

## Dependencies
- None (can execute independently)

## Wave
Wave 1 (parallel with Plan 3.2)

---

## Tasks

### Task 1: Install and configure Tailwind CSS with RTL support
**Files**: `mizan/frontend/package.json`, `mizan/frontend/tailwind.config.js`, `mizan/frontend/postcss.config.js`, `mizan/frontend/src/index.css`, `mizan/frontend/src/main.tsx`

**Step 1 — Install:**
```bash
cd mizan/frontend
npm install -D tailwindcss@3 postcss autoprefixer
npx tailwindcss init -p
```

**Step 2 — Configure `tailwind.config.js`:**
```js
/** @type {import('tailwindcss').Config} */
export default {
  content: ['./index.html', './src/**/*.{js,ts,jsx,tsx}'],
  theme: {
    extend: {
      fontFamily: {
        tajawal: ['Tajawal', 'sans-serif'],
      },
      colors: {
        mizan: {
          navy: '#1a1a2e',
          surface: '#f9f9f9',
        },
      },
    },
  },
  plugins: [],
}
```

**Step 3 — Create `src/index.css`:**
```css
@tailwind base;
@tailwind components;
@tailwind utilities;

html {
  direction: rtl;
}

body {
  font-family: 'Tajawal', sans-serif;
}
```

**Step 4 — Import in `main.tsx`:**
```tsx
import './index.css'
```

**Verify:** `npm run dev` starts without errors. Tailwind classes work (test with a temporary `className="text-red-500"` on any element).

### Task 2: Create shared Layout component
**File**: `mizan/frontend/src/components/Layout.tsx`

Create a layout shell that wraps all authenticated pages:
- Header: Mizan logo text (`ميزان`), user name, logout button — navy background (`bg-mizan-navy`)
- Navigation bar: المرصد، مدقق التحيز، التدريب links + admin link (conditional on `user.role`)
- Active route highlighting using `useLocation()` from react-router-dom
- Main content area: `{children}` slot with max-width container
- All styled with Tailwind utility classes

Props:
```tsx
interface LayoutProps {
  children: React.ReactNode
}
```

Use `useAuth()` for user info and `useLocation()` for active link detection.

### Task 3: Integrate Layout into App.tsx and refactor Dashboard
**Files**: `mizan/frontend/src/App.tsx`, `mizan/frontend/src/pages/Dashboard.tsx`

**App.tsx:** Wrap all protected routes in `<Layout>`. Remove the `Placeholder` component.

Updated structure:
```tsx
<Route path="/" element={<ProtectedRoute><Layout><Dashboard /></Layout></ProtectedRoute>} />
<Route path="/train" element={<ProtectedRoute><Layout><Placeholder label="التدريب" /></Layout></ProtectedRoute>} />
// etc. (train placeholder replaced in Plan 3.4)
```

**Dashboard.tsx:** Remove all inline styles and the header/nav (now in Layout). Replace with Tailwind utility classes. Dashboard becomes just the welcome message + 3 cards grid.

### Task 4: Verify RTL rendering
Run `npm run dev` and confirm in browser:
- All text renders right-to-left
- Navigation flows from right to left
- Tajawal font loads correctly
- No LTR bleed on any element
- `npm run build` succeeds without TypeScript errors

---

## Success Criteria
1. `npm run dev` starts without errors with Tailwind active
2. All existing pages render correctly with Tailwind replacing inline styles
3. Arabic text is RTL with Tajawal font throughout
4. Shared Layout component is used by all protected routes
5. `npm run build` succeeds without TypeScript errors
